	- show plot                     # Menampilkan grafik

	* Buat tes Signal Generator

"""
Input AM DSBFC
  - message x amp a fs b freq c         # Generate isyarat pemodulasi m(t)
  - carrier x amp a fs b freq c         # Generate isyarat pembawa c(t)
  - mul(message,carrier) as *           # Modulasi DSBFC -- (mt*ct)
  - add(*,carrier) as modulated         # Modulasi DSBFC -- (mt*ct)+ct                            
  - add envelope as demodulated         # Demodulasi DSBFC -- deteksi envelope
  - show plot                           # Menampilkan grafik hasil demodulasi
  a = Amplitudo                         c = Frekuensi isyarat
  b = Frekuensi sampling                x = jenis gelombang yang mau digunakan

	* Buat tes AM DSBFC
message cosine amp 1 fs 700 freq 5
carrier cosine amp 1 fs 700 freq 50
mul(message,carrier) as x
add(x,carrier) as modulated
add envelope as demodulated
show dsbfc timeDomain/frequencyDomain
show message/carrier/modulated/envelope/demodulated timeDomain/frequencyDomain
"""
"""
Input AM DSBSC
  - message x amp a fs b freq c         # Generate isyarat pemodulasi m(t)
  - carrier x amp a fs b freq c         # Generate isyarat pembawa c(t)
  - mul(message,carrier) as modulated   # Modulasi DSBSC -- (mt*ct)                        
  - mul(message,carrier,carrier) as **  # Demodulasi DSBSC -- (mt*ct*ct)
  - div(**,carrier.amp)                 # Demodulasi DSBSC -- (mt*ct*ct)/Ac
  - add filter order y cutoff z         # Demodulasi DSBSC -- filtering
  - show plot                           # Menampilkan grafik hasil demodulasi
  a = Amplitudo                         c = Frekuensi isyarat
  b = Frekuensi sampling                x = jenis gelombang yang mau digunakan
  y = order dari filter                 z = Frekuensi cutoff filter
	
	* Buat tes AM DSBSC
message cosine amp 1 fs 700 freq 5
carrier cosine amp 1 fs 700 freq 50
mul(message,carrier) as modulated
mul(message,carrier,carrier) as x
div(x,carrier.amp)
add filter order 50 cutoff 0.01
show dsbsc timeDomain/frequencyDomain
show message/carrier/modulated/demodulated/filtered timeDomain/frequencyDomain
"""
"""
Input AM SSB
  - message x amp a fs b freq c         # Generate isyarat pemodulasi m(t)
  - carrier x amp a fs b freq c         # Generate isyarat pembawa c(t) cos dan sin
  - hilbert x amp a fs b freq c         # Generate isyarat Hilbert(?)
  - mul(message,carrier.cos) as A       # Generate a = mt*ct(cos)
  - mul(hilbert,carrier.sin) as B       # Generate b = mt*ct(cos)
  - sub(A,B) as usb                     # Generate USB = a-b
  - add(A,B) as lsb                     # Generate LSB = a+b
  - mul(2,usb,carrier.cos)              # Demodulasi SSB -- (2*usb*ct(cos))
  - add filter order y cutoff z         # Demodulasi SSB -- filtering
  - show plot                           # Menampilkan grafik hasil demodulasi
  a = Amplitudo                         c = Frekuensi isyarat
  b = Frekuensi sampling                x = jenis gelombang yang mau digunakan
	
	* Buat tes AM SSB
message cosine amp 1 fs 700 freq 5
carrier cosine amp 1 fs 700 freq 50
hilbert sine amp 1 fs 700 freq 5
mul(message,carrier.cos) as A
mul(hilbert,carrier.sin) as B
sub(A,B) as usb
add(A,B) as lsb
mul(2,usb,carrier.cos)
add filter order 50 cutoff 0.01
show ssb timeDomain/frequencyDomain
show message/carrier/modulated/demodulated/filtered timeDomain/frequencyDomain
"""
"""
Input FM Modulation
  - message x amp a fs b freq c         # Generate isyarat pemodulasi m(t)
  - carrier x amp a fs b freq c         # Generate isyarat pembawa c(t)
  - modulation-index d                  # Input indeks modulasi
  - add integrator                      # Modulasi FM -- Menambahkan integrator
  - add phase-modulator as modulated    # Modulasi FM -- Menambahkan phase modulator
  - add differentiator                  # Demodulasi FM -- diskriminasi frekuensi
  - add envelope as demodulated         # Demodulasi FM -- deteksi selubung
  - show plot                           # Menampilkan grafik hasil demodulasi
  a = Amplitudo                         c = Frekuensi isyarat, d = indeks modulasi
  b = Frekuensi sampling                x = jenis gelombang yang mau digunakan
	
	* Buat tes FM Modulation
message cosine amp 1 fs 44100 freq 10
carrier cosine amp 1 fs 44100 freq 80
modulation-index 0.25
add integrator
add phase-modulator as modulated
add differentiator
add envelope as demodulated
show fm timeDomain/frequencyDomain
show message/carrier/modulated/demodulated timeDomain/frequencyDomain
"""

AM DSBFC biasa juga disebut dengan envelope AM karena jenis inilah yang menjadikan variasi tegangan pada sinyal pemodulasi sebagai envelope atau selubungnya. Proses modulasinya dilakukan dengan mengalikan antara pembawa dengan pemodulasi. Pengalian dapat dilakukan menggunakan pengali non linier yang akan menghasilkan gelombang AM DSBFC dan ditambah dengan hasil sampingan. Agar hanya hasil yang diinginkan yang diperoleh, dilakukan penapisan agar sinyal hasil sampingan dari pengali dapat dihilangkan.

Berikut kode yang dapat digunakan pada modul ini

message wave amplitude a freq-sampling b frequency c
carrier wave amplitude a freq-sampling b frequency c
multiply(message,carrier) as x
add(x,carrier) as modulated
add envelope as demodulated
show plot domain
Hal yang harus diperhatikan dalam menuliskan kode
'wave' harus diisi dengan jenis gelombang isyarat, meliputi: sine/cosine/square/sawtooth
'a' harus diisi dengan nilai amplitudo isyarat
'b' harus diisi dengan nilai frekuensi pencuplikan
'c' harus diisi dengan nilai frekuensi isyarat
'plot' harus diisi dengan data yang ingin ditampilkan, meliputi: message/carrier/modulated/envelope/demodulated
Jika ingin menampilkan seluruh data, isi 'plot' dengan dsbfc
'domain' harus diisi dengan ranah yang ingin ditampilkan, meliputi: timeDomain atau frequencyDomain
